◆ Exists

Boolean Exists
get

Returns true if the path exists in the AssetDatabase.

This may still return true for asset files that have been deleted externally if those changes have not been imported via CodeSmile.Editor.Asset.Database.ImportAll.

In Unity 2023.2 or newer uses the new AssetPathExists method. In earlier versions AssetPathToGUID is used.
