Imports a file at a given path that was created or modified 'externally'. Externally refers to any means other than AssetDatabase methods such as System.IO or batch scripts.

CodeSmile AssetDatabase provides convenience Create methods that automatically import assets: CodeSmile.Editor.Asset.File.Create(Byte[],Path) CodeSmile.Editor.Asset.File.Create(string,Path)
