static T Load< T > ( Path  path)
static

Loads an asset at path.

Will only load visible sub-objects. Returns the first object of the type found. If there are multiple objects of the same type, use CodeSmile.Editor.Asset.SubAsset.LoadVisible instead.

Parameters
pathPath to an asset file.
Template Parameters
TType of the asset.
Returns
The loaded asset object, or null if the asset does not exist or does not contain a visible object of type T.
