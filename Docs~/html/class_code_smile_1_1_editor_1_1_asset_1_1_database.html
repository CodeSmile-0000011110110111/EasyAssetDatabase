static void AllowAutoRefresh ()
 Will stop Unity from automatically importing assets. Must be called in pair with DisallowAutoRefresh. Multiple calls must be matched with an equal number of calls to DisallowAutoRefresh since internally this is using a counter that needs to return to 0 before auto refresh is going to be enabled again. Note: Has no effect if Preferences => Asset Pipeline => Auto Refresh is disabled to begin with. Same as AssetDatabase.AllowAutoRefresh(). DisallowAutoRefresh
