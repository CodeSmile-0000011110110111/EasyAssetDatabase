◆ SaveAll()

static void SaveAll ( )
static

Saves all unsaved (dirty) assets.

CAUTION: Consider that a user generally does NOT want an editor script to 'randomly' save all unsaved assets! Use only where absolutely appropriate, otherwise prefer to use CodeSmile.Editor.Asset.File.Save to explicitly save each modified asset. Preferably do so within a CodeSmile.Editor.Asset.File.BatchEditing operation. Just be nice to fellow Editor users. ;)
