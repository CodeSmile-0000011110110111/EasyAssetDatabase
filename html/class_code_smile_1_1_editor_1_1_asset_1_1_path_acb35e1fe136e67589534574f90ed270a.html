static Boolean CodeSmile.Editor.Asset.Path.IsValid ( String  path)
static

Returns true if the provided path is valid. This means it contains no illegal folder or file name characters and it isn't too long. If this returns false, Asset.GetLastErrorMessage() contains more detailed information. Asset.GetLastErrorMessage()

Parameters
path
Returns
True if the string is a valid path and contains no illegal characters for a path or file. False in all other cases.
